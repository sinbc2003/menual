# 631쪽

## 4세대 나이스 호봉 작업 방법 (계속)

### 가. 초임호봉 관리 (계속)

#### 3) 초임호봉대상자 조회 및 자격/경력 입력 (계속)

##### ◎ 전임경력신청정보 (계속)

| 항목 | 설명 |
|:---|:---|
| **⑩ {경력제외기간}** | 휴직 등의 사유로 경력기간에서 제외되는 기간 입력(경력기간을 입력하여야 선택 버튼을 클릭하여 입력할 수 있음) |
| **⑪ {경력인정기간}** | 기초정보 입력 시 자동 계산됨 |
| **⑫ {증빙 첨부}** | 호봉획정 근거 자료 첨부 가능(첨부하지 않아도 다음 메뉴 진행 가능) |
| **⑬ {근무년수 인정여부}** | 선택 시 근무년수에 반영됨 |

---

##### ◎ 군경력신청정보

**작업순서**: {군경력신청정보} 탭 선택 – {추가} - 자료 입력 - {저장}

| 항목 | 설명 |
|:---|:---|
| **③ {인정여부}** | 선택 시 해당 경력이 호봉경력으로 반영 |
| **④ {인정률}** | 교육공무원등의 경력환산율표에 따라 인정되는 경력환산율을 입력 |
| **⑤ {군별명}** | 병적증명서상의 군별 선택 |
| **⑥ {복무형태}** | 병적증명서상의 역종 선택 |
| **⑦ {의무복무여부}** | 의무복무일 경우 선택 |
| **⑧ {계급}** | 병적증명서상의 계급 선택 |
| **⑨ {군경력}** | 병적증명서상의 복무기간 입력 |
| **⑩ {경력제외기간}** | 군 복무중 경력기간에서 제외되는 기간 입력(경력기간을 입력하여야 선택 버튼을 클릭하여 입력할 수 있음) |
| **⑪ {경력인정기간}** | 기초정보 입력 시 자동 계산됨 |
| **⑫ {증빙 첨부}** | 호봉획정 근거 자료 첨부 가능(첨부하지 않아도 다음 메뉴 진행 가능) |
| **⑬ {근무년수 인정여부}** | 선택 시 근무년수에 반영됨 |

| 항목 | 설명 |
