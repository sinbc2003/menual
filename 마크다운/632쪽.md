# 632쪽

## 4세대 나이스 호봉 작업 방법 (계속)

### 가. 초임호봉 관리 (계속)

#### 3) 초임호봉대상자 조회 및 자격/경력 입력 (계속)

##### ◎ 대학학력신청정보 (계속)

| 항목 | 설명 |
|:---|:---|
| **⑥ {학교명}** | 졸업증명서상 학교명 입력 |
| **⑦ {학교구분}** | 졸업증명서상 졸업 학교 선택 |
| **⑧ {편입}** | 여부 선택 시 편입 학년 입력 가능 |
| **⑨ {수학기간}** | 대상자의 입학일과 졸업일을 입력 |
| **⑩ {수학제외기간}** | 휴학 등의 사유로 경력기간에서 제외되는 기간 입력(경력기간을 입력하여야 선택 버튼을 클릭하여 입력할 수 있음) |
| **⑪ {수학인정기간}** | 기초정보 입력 시 자동 계산됨 |
| **⑫ {전공학과명}** | 졸업증명서상 학과명 입력(선택 사항) |
| **⑬ {증빙 첨부}** | 호봉획정 근거 자료 첨부 가능(첨부하지 않아도 다음 메뉴 진행 가능) |

- 독학사 및 학점은행제 학위도 같은 방법으로 등록

---

##### ◎ 대학원학력신청정보

> ※ 임용 전 대학원에서 학위를 취득한 경력은 '대학원학력신청정보' 탭이 아닌 '전임경력신청정보' 탭에서 입력하여야 함

| 항목 | 설명 |
|:---|:---|
| **③ {인정여부}** | 선택 시 해당 경력이 호봉경력으로 반영 |
| **④ {인정률}** | 교육공무원등의 경력환산율표에 따라 인정되는 경력환산율을 입력 |
| **⑤ {학위구분}** | 학위수여증명서상 학위명 입력 |
| **⑥ {학교명}** | 학위수여증명서상 학교명 입력 |
| **⑦ {최저수업년한}** | 학칙상 해당 대학원의 최저수업년한 입력 |
| **⑧ {계절제/야간제 구분}** | 해당 대학원의 계절제/야간제 해당 여부 선택 |
| **⑨ {수학기간}** | 대상자의 입학일과 졸업일을 입력 |
| **⑩ {수학제외기간}** | 휴학 등의 사유로 경력기간에서 제외되는 기간 입력(경력기간을 입력하여야 선택 버튼을 클릭하여 입력할 수 있음) |
| **⑪ {수학인정기간}** | 기초정보 입력 시 자동 계산됨 |
| **⑫ {증빙 첨부}** | 호봉획정 근거 자료 첨부 가능(첨부하지 않아도 다음 메뉴 진행 가능) |
