## NEIS 교원인사관리 (계속)

## 3. NEIS 교원인사 권한부여

### 가. 권한부여 흐름

1) 교원인사 권한은 반드시 경기도교육청이나 교육지원청 등의 인사권한 관리자를 통해서만 부여 가능

2) 권한 관리자는 필요 이상의 권한이 부여되지 않도록 수시로 권한부여 상태를 점검

3) 시스템관리의 권한관리 외에도 인사기록탭관리, 인사권한등록에서도 올바른 내용을 적용하여야 함 **(최소한의 업무에 필요한 내용을 적용)**

---

**[권한부여 흐름도]**

```
시도교육청
    └── 경기도교육청 기관마스터
            └── 학교급별 교원인사 단위업무권한관리자 지정
                    ├── 교원인사과(공립학교) - 인사권한관리자
                    │       ├── 공립 초등 → 사용자
                    │       └── 공립 중등 → 사용자
                    │
                    └── 유아교육과(유치원), 특수교육과(특수), 사립학교지원과(사립)
                            └── 인사권한관리자
                                    ├── 유치원 → 사용자
                                    ├── 특수 → 사용자
                                    └── 사립 → 사용자

교육지원청
    ├── 공립 초등 권한관리자 → 초등학교 사용자
    └── 공립 중등 권한관리자 → 중학교, 고등학교 사용자
            └── 학교급별 권한관리자 지정 → 유치원, 특수학교, 사립학교 사용자
```

**범례**
- ─► 단위업무별권한관리자 지정
- ···► 사용권한 부여
