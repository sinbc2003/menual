# 633쪽

## 4세대 나이스 호봉 작업 방법 (계속)

### 가. 초임호봉 관리 (계속)

#### 3) 초임호봉대상자 조회 및 자격/경력 입력 (계속)

##### ◎ 교원자격증신청정보

**작업순서**: {교원자격증신청정보} 탭 선택 – {추가} - 자료 입력 - {저장}

| 항목 | 설명 |
|:---|:---|
| **③ {인정여부}** | 선택 시 해당 교원자격이 호봉경력(기산호봉)으로 반영 |
| **④ {교원자격명}** | 교원자격증상 자격명 선택. 선택 자격에 따라 기산호봉 산정됨 |
| **⑤ {발급일자}** | 교원자격증상 발급일자 입력 |
| **⑥ {표시과목}** | 교원자격증상 표시과목 선택 |
| **⑦ {발급기관}** | 교원자격증상 발급기관 입력 |
| **⑧ {자격번호}** | 교원자격증상 자격번호 입력 |
| **⑨ {증빙 첨부}** | 호봉획정 근거 자료 첨부 가능(첨부하지 않아도 다음 메뉴 진행 가능) |

---

##### ◎ 가산정보

**작업순서**: {가산정보} 탭 선택 – {추가} - 자료 입력 - {저장}

| 항목 | 설명 |
|:---|:---|
| **{가산항목}** | 공무원보수규정에 따른 가산항목 중 대상자에게 적용되는 항목 선택 |
| **{가산연수}** | 대상자의 가산 연수 입력(자동계산 아님에 유의) |

**가산항목 종류:**
- 선택
- 사범계졸업
- 특수학교(학급)
- 휴업승급
