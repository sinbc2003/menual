# 669쪽

## NEIS 교원인사관리 (계속)

### 3. NEIS 교원인사 권한부여 (계속)

#### 라. 권한부여 방법

##### 1) 개인별권한등록

**가) 메뉴** : [시스템관리] - [사용자권한관리] - [개인별권한등록]

**나) 방법** : 우측상단 [나의 업무]에서 경기도*교육청 선택 -> [개인별권한등록] 클릭 -> 사용자명 찾기/조회

① **업무그룹 등록** : {업무그룹등록} 버튼 클릭 -> 권한 부여할 업무그룹명 선택 후, 저장

**[업무그룹 추가 화면 절차]**

1. 업무영역: 업무영역 선택
2. 대분류: 전체 선택
3. 업무그룹등록조직: 전체 선택
4. 업무그룹명: 검색어 입력
5. {조회} 클릭
6. 업무그룹 목록에서 부여할 업무그룹 선택 (체크박스)
7. {저장} 클릭
