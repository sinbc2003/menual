## 2. 근무성적평정 관련 서식 (계속)

### ※ 정량평가 평가사항 _ 교사용

| 구분 | 평가사항 | 평가요소 | 평가지표 |
|------|------|------|------|
| 교사 | 근무실적 및 근무수행능력 | 가. 학습지도<br>(30점) | 1) 주당 수업시간<br>2) 수업공개(연구수업 및 학부모 공개수업 등) 횟수<br>3) 교내외 수업컨설팅 횟수<br>4) 학습지도 곤란도(통합수업 지도/복식학급 지도/학습부진아 지도/다학년 지도/다교과 지도/보강수업) |
| | | 나. 생활지도<br>(30점) | 1) 학생 또는 학부모 상담실적<br>2) 생활지도 곤란도(담임/비담임)<br>3) 학년 곤란도 |
| | | 다. 전문성개발<br>(10점) | 1) 연수이수 실적<br>2) 수업동아리 및 교과연구회 활동 실적<br>3) 연구대회 및 연구개발 실적 |
| | | 라. 담당업무<br>(30점) | 1) 업무 곤란도(다문화·탈북학생 지원/방과후학교 운영/자유학기제 운영)<br>2) 업무 추진 실적(보직/학년대표/교과대표/일반교사) |

| 비고 |
|------|
| ※ 다면평가관리위원회는 필요한 경우 평가지표를 추가 또는 삭제하거나 수정할 수 있다. |
