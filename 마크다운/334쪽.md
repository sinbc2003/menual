# 334쪽

## 마. 공무외의 국외여행 (계속)

### 공무외국외여행 나이스 상신 방법 (계속)

---

**[예시]**

▣ 2025.8.7.(목) 05:00에 태국으로 출국하여 2025.8.11.(월) 05:00에 입국 후, 2025.8.11.(월) 09:00에 출근하여 근무 또는 교육공무원법 제41조 연수 사용하는 경우

| 구분 | 종별 | 기간 | 사유 |
|------|------|------|------|
| 한번에 상신 | 연가 (2일) | 2025.8.7.(목) 05:00<br>~2025.8.10.(일) 24:00 | 태국(여행기간: 2025.8.7.(목) 05:00<br>~2025.8.11.(월) 05:00) |
| 나누어 상신 | 연가 (2일) | 2025.8.7.(목) 05:00<br>~2025.8.8.(금) 24:00 | 태국(여행기간: 2025.8.7.(목) 05:00<br>~2025.8.11.(월) 05:00) |
| | 기타 (2일) | 2025.8.9.(토) 00:00<br>~2025.8.11.(일) 24:00 | |

**'연가'는 시간단위가 아닌 일단위 근무상황으로 2025.8.11.(월) 05:00까지 연가를 상신하면 복무상신시 사용일수를 조정하더라도 8.11은 연가를 사용한 것으로 반영됨에 유의**

---

▣ 2025.8.6.(수) 근무 후 22:00에 태국으로 출국하여 2025.8.11.(월) 05:00에 입국하고, 2025.8.11.(월)에 근무하지 않는 경우

| 구분 | 종별 | 기간 | 사유 및 비고 |
|------|------|------|------|
| 한번에 상신 | 연가 (3일) | 2025.8.7.(목) 00:00<br>~2025.8.11.(월) 17:00 | 태국(여행기간: 2025.8.6.(수) 22:00<br>~2025.8.11.(월) 05:00) |
| 나누어 상신 | 연가 (2일) | 2025.8.7.(목) 00:00<br>~2025.8.8.(금) 24:00 | 태국(여행기간: 2025.8.6.(수) 22:00<br>~2025.8.11.(월) 05:00) |
| | 기타 (2일) | 2025.8.9.(토) 00:00<br>~2025.8.10.(일) 24:00 | |
| | 연가 (1일) | 2025.8.11.(월) 00:00<br>~2025.8.11.(월) 17:00 | |
